## Features

- **Interactive gallery**: An automated, folder-based timeline that showcases your crochet journey with smooth shimmer effects and lazy loading.
- **Dynamic patterns**: A powerful MDX-driven pattern library. Just drop a folder in, and the site does the rest.
- **Glassmorphism design**: A soft, modern aesthetic with vibrant pink accents and premium typography.

---

## Quick start

Ensure you have [Node.js](https://nodejs.org/) installed, then follow these steps:

1. **Install dependencies**:

   ```bash
   npm install
   ```

2. **Run the development server**:

   ```bash
   npm run dev
   ```

3. **Open the site**:
   Navigate to [http://localhost:5173](http://localhost:5173) in your browser.
